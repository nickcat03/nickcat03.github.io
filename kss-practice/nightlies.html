Future Releases

Going forward, the most up-to-date versions can be found in the GitHub repository.

12-23-23

Download
  • Milky Way Wishes abilities can be automatically obtained / removed when entering a level (depending on the category being practiced).
    • On planet select, press Select to scroll through each category:
    • Any%
    • Any% Plasma
    • 100%
    • Toggle off (default)
    • Visual feedback still needs to be added.
  • Planets can now be quick selected with L or R.
  • QSQL now copies SA-1 RAM, allowing for more accurate saving and loading. VRAM transfers are still a work in progress.

12-21-23

Download
  • Committing to a complete rewrite of the entire practice hack, which is now uploaded onto GitHub. This will allow for easier management and organization, as writing directly to the ROM file itself was very time consuming.
  • There are several things that were added and removed in this hack. All the removed features will eventually come back, but I just need time to add them in again.
  • Added:
    • Save State support (QSQL), use Select+L to save and Select+R to load. These hotkeys may be changed later.
    • Every single ability can now be quick selected:
      • Hold R + A + ...
      • Up: Jet
      • Right: Wheel
      • Down: Hammer
      • Left: Plasma
      • Hold L + A + ...
      • Up: Wing
      • Right: Stone
      • Down: Suplex
      • Left: Cutter
      • Hold L + R + A + ...
      • Press Left or Right to cycle through all abilities.
    • Files are deleted instantly when prompted to.
    • Subgame menus will no longer play cutscenes when inactive.
    • Milky Way Wishes menu will always highlight "Continue" by default.
    • Very minor performance tweaks.
  • Removed:
    • Custom HUD. Given that most weren't too fond of its inclusion, I will only include it once I can make it completely optional (it will be a togglable setting in the game).
    • Music toggle.
    • Custom respawn coordinates were removed as QSQL in tandem with instant death makes them obsolete. I will put in my best effort to make QSQL as stable as possible.
  • Any other features that were in the ROM previously are included in this rewrite.
