Super Scribblenauts Speedrun Guide (Any%)

Written by Nippo, with strats taken from SoapAgent, KitLemonfoot, and Hagata

How to use this guide:

Super Scribblenauts is a fairly easy speedrun to follow, given that majority of the run involves spawning specific items into the world.

The best way to run this game is by playing through the levels one by one while keeping this guide open to reference what to spawn in each level.

If you end up finding developments in the run which you deem to be faster, feel free to send me a message so I can update it accordingly.

Speedrun Prep

Super Scribblenauts has an adjective book , which saves your adjective history for use later. This saves time with adjectives that take a while to type.

In addition, the game stores 7 previous nouns used by pressing the undo button . Nouns can be stored before the run starts to save a bit of time on the first two levels.

Below are the list of adjectives and nouns that should be saved.

Adjectives

  • Blue
  • Yellow
  • Red
  • Green
  • Supersonic
  • Sonic
  • Unavoidable
  • Ridable
  • Wee
  • Dragonish
  • Surgical
  • Eternal

Nouns

  • Emu
  • Tire

History can be cleared by resetting the game.

Below is the fastest way to get the above words stored in history. Note that the adjectives are split between the two nouns as nouns can only have a maximum of 10 adjectives (this can be done in the sandbox screen before file select, it is persistent across files).

  • Red Green Supersonic Sonic Unavoidable Ridable Wee Dragonish Surgical Eternal Tire

When an adjective is used, the word is pushed to the front of the adjective book.

Blue, Yellow, Red, and Green

These four colors are used in one of the early levels in the game, so it is best to place them first in the book.

The unavoidable sonic ridable wee oak

An "unavoidable sonic ridable wee oak" is used to break collision in many levels. This allows Maxwell to go through movable doors and deadly obstacles that would otherwise kill him. Pretty much the entirety of World S1 and S2 abuses this exploit.

The easiest way to hop on the tree is by dragging it on top of Maxwell, and he should start riding it immediately.

It is best to watch videos of these levels to get an understanding of the optimal movement in them.

Why use "supersonic" and "sonic"?

Though these two adjectives achieve the same result, there are times where having two different adjectives makes it easier to select them in the adjective book.

Throughout the run, "sonic" is used in tandem with "unavoidable", "wee", and "ridable" when spawning the oak. However, there are times where "sonic" is used by itself with wings, which ends up separating this word from the rest of the adjectives it is used with.

By using two adjectives that have the same purpose, we can use "supersonic" on its own while keeping "sonic" adjacent to "unavoidable", "wee", and "ridable" in the adjective book.

Dragonish, surgical, and eternal

These last few adjectives are used in late-game, so they are placed in the back of the book.

Why not add more nouns?

Adding more nouns requires the undo button to be pressed 3+ times, which ends wasting time than just typing the words. If it is faster for you, feel free to add any additional nouns for later use.
